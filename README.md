# 📚 Anki-tecture (Anki 자동 생성기)

**Anki-tecture**는 학습 자료(PDF)를 분석해 Anki 카드를 자동으로 생성하고, 내 Anki 데스크톱 앱에 직접 추가해 주는 자동화 툴입니다. 

Google Gemini API(Flash)의 텍스트 구조화 및 시각 분석 능력을 활용하여 문서 내의 문제, 해설은 물론 이미지 좌표를 추적해 실제 사진까지 깔끔하게 잘라내어 카드에 연동합니다.

---

## 🛠 1단계: 사전 준비 사항 (한 번만 하면 됩니다.)

이 프로그램을 실행하려면 컴퓨터에 다음 4가지가 미리 준비되어 있어야 합니다. 

### 1. Python 3.8 이상 설치
파이썬이 깔려있지 않다면 아래 순서대로 설치해 주세요.
1. [파이썬 공식 홈페이지](https://www.python.org/downloads/)에 접속하여 노란색 Download 버튼을 눌러 설치 파일을 받습니다.
2. 다운받은 설치 파일을 실행합니다.
3. **[주의!]** 설치 첫 화면 맨 아래에 있는 **`Add python.exe to PATH` (또는 `Add Python 3.x to PATH`) 네모 박스를 꼭 체크**해야 합니다. 
4. `Install Now`를 눌러 설치를 완료합니다.

### 2. Anki 데스크톱 프로그램 설치
1. [Anki 공식 홈페이지](https://apps.ankiweb.net/)에서 프로그램을 다운로드하고 설치합니다.
2. 설치가 끝나면 Anki 프로그램을 켜둡니다.

### 3. AnkiConnect 애드온 설치 
1. 켜져 있는 Anki 프로그램 상단 메뉴에서 `도구(Tools)` -> `애드온(Add-ons)` -> `애드온 얻기(Get Add-ons)`를 누릅니다.
2. 코드 입력창에 **`2055492159`** 를 입력하고 확인을 누릅니다.
3. 설치가 완료되면 Anki 프로그램을 완전히 껐다가 **다시 켜주세요.**

### 4. Google Gemini API Key 발급
1. [Google AI Studio](https://aistudio.google.com/)에 구글 아이디로 로그인합니다.
2. `Get API key` 버튼을 눌러 새로운 API 키(길고 복잡한 영어+숫자 조합)를 생성하고 메모장 등에 복사해 둡니다. (무료입니다!)

## 🚀 2단계: 설치 및 세팅 방법

### 1. 프로그램 다운로드
1. 이 깃허브 페이지 오른쪽 위에 있는 초록색 `<> Code` 버튼을 누릅니다.
2. `Download ZIP`을 클릭하여 파일을 다운받습니다.
3. 바탕화면이나 원하는 곳에 **압축을 풀어줍니다.** (폴더 이름이 `anki-tecture-main`으로 나올 겁니다.)

### 2. 명령 프롬프트(터미널) 열기
이제 컴퓨터에게 명령을 내릴 까만 창을 열어야 합니다.
* **Windows:** 압축을 푼 폴더 안으로 들어간 뒤, 창 맨 위쪽의 **폴더 경로(주소창)를 클릭하고 `cmd` 라고 친 후 엔터**를 누릅니다. 그러면 현재 폴더 위치에서 까만 창이 바로 열립니다.
* **Mac:** 압축을 푼 폴더를 우클릭하고 '폴더에서 새로운 터미널 탭 열기'를 누릅니다.

### 3. 가상환경(전용 작업실) 만들고 켜기
컴퓨터가 꼬이지 않게 이 프로그램만을 위한 독립된 방을 만듭니다. 열려있는 까만 창에 아래 명령어를 한줄씩 복사해서 붙여넣고 엔터를 치세요.

**Windows 사용자:**
```cmd
python -m venv venv
venv\Scripts\activate
```
**Mac 사용자:**
```Bash
python3 -m venv venv
source venv/bin/activate
```
(성공하면 입력창 맨 앞에 (venv)라는 글자가 생깁니다. 앞으로 프로그램을 켤 때는 항상 이 글자가 보여야 합니다!)

### 4. 필요한 재료(라이브러리) 설치하기
(venv)가 켜진 상태에서 아래 명령어를 쳐서 필요한 도구들을 한 번에 설치합니다. (시간이 조금 걸릴 수 있습니다.)

```Bash
pip install streamlit google-genai pymupdf requests python-dotenv
```

### 5. API 키 입력하기 (보안 설정)
1. 압축을 푼 폴더 안에 마우스 우클릭을 해서 **새 텍스트 문서(메모장)**를 하나 만듭니다.

2. 파일 이름을 .env 로 변경합니다. (앞에 점 .이 꼭 있어야 하고, .txt 같은 확장자가 없어야 합니다. 경고창이 뜨면 '예'를 누르세요.)

3. 만든 .env 파일을 메모장으로 열고, 아까 발급받은 API 키를 아래처럼 적은 뒤 저장합니다.

```Plaintext
GEMINI_API_KEY="여기에_발급받은_API_KEY_입력"
```

### 6. Anki 덱 만들기
카드를 추가할 덱을 만들어주세요! 
Anki에 들어가면 보이는 '덱 만들기' 버튼을 클릭해서 만들면 됩니다. 


## 💻 3단계: 사용 방법 (매번 실행할 때마다)
초기 세팅이 끝났습니다! 이제 안키를 만들고 싶을 때마다 아래 순서대로 실행하세요.

1. 먼저 컴퓨터에서 Anki 프로그램을 실행해 둡니다.

2. 프로그램 폴더에서 주소창에 cmd를 쳐서 까만 창을 열고, 가상환경을 켭니다. 
    가상환경 켜는 법: Windows라면 'venv\Scripts\activate' 입력, Mac이라면 'source venv/bin/activate' 입력하기
    (입력창 앞에 (venv)가 없다면 2단계의 3번 과정을 다시 쳐서 켜주세요.)

3. 까만 창에 아래 명령어를 입력하고 엔터를 칩니다.

```Bash
streamlit run app.py
```

4. 자동으로 열리는 인터넷 브라우저 창에서 다음 기능들을 수행할 수 있습니다:
- 분석할 족보 또는 요약본 PDF 파일 업로드 (여러 개 동시 가능)
- 자료 성격 선택 (형태에 따라 최적화된 프롬프트가 적용됩니다)
- 카드를 추가할 대상 Anki 덱 선택 또는 새 덱 이름 입력

5. 실행 버튼 클릭!


## ⚙️ 주요 기능 작동 원리
- 프론트엔드 (app.py): Streamlit을 이용해 파일을 메모리(Bytes)에 올리고 옵션을 선택하는 UI 제공.
- 백엔드 처리 (work.py): 메모리의 PDF 바이트를 Gemini API로 전송.
- 이미지 자동 크롭: Gemini가 PDF 내 이미지의 Bounding Box 좌표(0~1000)를 반환하면, PyMuPDF(fitz)가 해당 좌표를 실제 크기로 환산해 고화질로 잘라냄.
- 다이렉트 Anki 추가: AnkiConnect의 storeMediaFile과 addNote 액션을 이용해 생성된 텍스트와 이미지 미디어를 로컬 Anki로 즉각 쏴줌.
